## Release 3.0.0 - 2026-03-17
- Added the new Blueprints mod.
- Added a `BLUE` action in PDA Orders under `Crew Orders & Building`.
- Added in-PDA blueprint selection with file-name display and a `Select Blueprint` button for loading saved layouts into placement mode.
- Added blueprint cursor/icon support and runtime-built PDA UI that survives panel refreshes.
- Added capture and placement support for saved part rotation so placed layouts keep their intended facing.
- Added overlay-aware install resolution so blueprint placement uses the correct placeholder/installable variants.
- Added a config toggle to disable writing blueprint JSON files while keeping capture mode available.
- Added cancel handling for blueprint mode via `Esc`, right-click, and PDA close.
- Updated build/release scripts for the consolidated multi-mod release flow.
